Configuration:

Obtenez une clé API Google Maps sur Google Cloud Console.

Remplacez 'YOUR_API_KEY' par votre clé API dans le fichier index.html .

Créez un Projet sur chargetrip.com.

Remplacez 'YOUR_CLIENT_ID' et 'YOUR_APP_ID' par votre Client ID et APP ID dans le fichier getVehicles.js .


Utilisation:

Lancez le serveur HTTP en utilisant http-server :
npx http-server

Lancez le serveur Python :
python app.py

Ouvrez votre navigateur web et accédez à l'adresse http://localhost:8080 pour utiliser l'application.

Sélectionnez votre véhicule électrique dans la liste.

Entrez votre ville de départ et votre destination.

Cliquez sur le bouton "Calculer l'itinéraire" pour obtenir votre trajet.
